# 현재 정규식 규칙 목록 및 추가 규칙 생성 요청

## 프로젝트 개요
은행 거래 내역 텍스트를 정규화하는 AI 시스템에서 사용할 정규식 규칙을 관리하고 있습니다. 현재 31개의 규칙이 있으며, 더 많은 규칙이 필요합니다.

## 현재 정규식 규칙 분석

### 카테고리별 분류

#### 1. 주유소 (12개)
- `(지에스칼텍스|에스케이엔크린|지에스25)\(하\)주` - 한글 표기 하행선 주유소 패턴
- `(지에스칼텍스|에스케이엔크린|지에스25)\(상\)주` - 한글 표기 상행선 주유소 패턴  
- `(.+?)\(상\)주` - 상행선 주유소 패턴 (예: GS칼텍스(상)주)
- `(.+?)\(하\)주` - 하행선 주유소 패턴 (예: SK엔크린(하)주)
- `(지에스칼텍스|에스케이엔크린|지에스25)` - 한글 표기 브랜드명 패턴
- `(GS칼텍스|SK엔크린|에쓰오일|현대오일뱅크|S-?OIL)셀프` - 셀프 주유소 패턴
- `(GS칼텍스|SK엔크린|에쓰오일|현대오일뱅크|S-?OIL)(?!셀프)` - 주유소 브랜드명 패턴
- `(.+?)주유소` - 일반 주유소 키워드 패턴

#### 2. 편의점 (5개)  
- `GS25` - GS25 편의점 패턴
- `\bCU\b` - CU 편의점 패턴
- `(세븐일레븐|7-?ELEVEN|7일레븐|세븐이레븐|SEVEN\s*ELEVEN)` - 세븐일레븐 편의점 패턴 (다양한 표기법)
- `(이마트24|EMART24)` - 이마트24 편의점 패턴
- `(미니스톱|MINISTOP)` - 미니스톱 편의점 패턴

#### 3. 온라인서비스 (7개)
- `(GODADDY|고대디)` - GoDaddy 온라인서비스 패턴
- `(GOOGLE|구글)` - Google 온라인서비스 패턴
- `(AMAZON|아마존|AWS)` - Amazon 온라인서비스 패턴
- `(NETFLIX|넷플릭스)` - Netflix 온라인서비스 패턴
- `(SPOTIFY|스포티파이)` - Spotify 온라인서비스 패턴
- `(YOUTUBE|유튜브)` - YouTube 온라인서비스 패턴

#### 4. 카센터 (3개)
- `(.+?)카센터` - 카센터 키워드 패턴
- `(현대|기아|벤츠|BMW|아우디|볼보)(?:자동차)?(?:서비스센터|서비스|정비)` - 자동차 브랜드 서비스센터 패턴
- `(.+?)정비소` - 정비소 키워드 패턴

#### 5. Privacy (개인정보 마스킹) (4개)
- `(\d{3})-(\d{4})-(\d{4})` → `$1-****-$3` - Phone number masking
- `(\d{4})-(\d{4})-(\d{4})-(\d{4})` → `$1-****-****-$4` - Credit card number masking
- `\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b` → `***@***.***` - Email masking
- `\b[가-힣]{2,4}\b` → `***` - Korean name masking

#### 6. Normalization (브랜드명 정규화) (3개)
- `\b스타벅스\b` → `커피전문점` - Brand name normalization
- `\b맥도날드\b` → `패스트푸드점` - Brand name normalization  
- `\b이마트\b` → `대형마트` - Brand name normalization

#### 7. Cleanup (텍스트 정리) (2개)
- `\s+` → ` ` - Multiple spaces to single space
- `^[\s\t\n\r]+|[\s\t\n\r]+$` → `` - Trim whitespace

## 요청사항

**다음 카테고리들에 대해 더 많은 정규식 패턴을 생성해주세요:**

### 1. 음식점/카페 (부족함 - 현재 3개만 있음)
- 한국의 주요 프랜차이즈 음식점, 카페 체인
- 지역 음식점 패턴
- 배달 서비스 관련 패턴

### 2. 쇼핑/유통 (부족함)
- 대형마트, 백화점, 쇼핑몰
- 온라인 쇼핑몰
- 의류, 화장품 브랜드

### 3. 교통 (없음)
- 대중교통 (지하철, 버스, 택시)
- 고속도로 통행료
- 주차장, 주차비

### 4. 의료/병원 (없음)
- 병원, 의원, 약국
- 치과, 안과 등 전문의료기관
- 건강검진센터

### 5. 금융 (없음)  
- 은행, ATM 수수료
- 보험료
- 증권, 투자 관련

### 6. 통신/인터넷 (부족함)
- 통신사 요금
- 인터넷 서비스
- 스트리밍 서비스 (현재 Netflix, Spotify, YouTube만 있음)

### 7. 교육 (없음)
- 학원, 교육기관
- 온라인 강의 플랫폼
- 서점, 문구점

### 8. 생활서비스 (없음)
- 세탁소, 미용실
- 택배, 배송
- 수리 서비스

## 패턴 생성 가이드라인

1. **우선순위**: 50-100 사이 (높을수록 먼저 매칭)
2. **카테고리**: 명확한 업종 분류
3. **설명**: 패턴의 목적을 명확히 기술
4. **다양한 표기법 고려**: 한글/영문, 띄어쓰기, 줄임말 등
5. **replacement**: 필요시 정규화된 형태로 변환

## 예시 형식

```json
{
  "pattern": "(스타벅스|STARBUCKS|스벅)",
  "replacement": "커피전문점",
  "description": "스타벅스 다양한 표기법 패턴",
  "category": "카페",
  "enabled": true,
  "priority": 90
}
```

**한국의 거래 내역에서 자주 등장하는 업체들을 중심으로 포괄적인 정규식 패턴들을 생성해주세요.**
