## 1. 🔴 보안 및 인증 시스템 (필수)

### 1.1 사용자 인증 체계 구축
**현재 상태**: 하드코딩된 userId=1 사용, 인증 없음
**필요 작업**:
- [ ] OAuth 2.0 소셜 로그인 구현 (Google, Apple, Kakao, Naver)
- [ ] JWT 토큰 기반 인증 시스템 완성
- [ ] Refresh Token 로직 구현
- [ ] 디바이스별 세션 관리
- [ ] 비밀번호 찾기/재설정 기능
- [ ] 2FA (Two-Factor Authentication) 지원

### 1.2 데이터 보안
**현재 상태**: 평문 저장, 암호화 없음
**필요 작업**:
- [ ] 민감 데이터 AES-256 암호화
- [ ] SSL/TLS 인증서 적용 (HTTPS)
- [ ] API Rate Limiting 구현
- [ ] SQL Injection 방어 코드 검증
- [ ] XSS/CSRF 공격 방어
- [ ] 앱 난독화 및 안티 디버깅

### 1.3 개인정보보호
**현재 상태**: 개인정보 처리 방침 없음
**필요 작업**:
- [ ] GDPR/개인정보보호법 준수
- [ ] 데이터 익명화 처리
- [ ] 사용자 데이터 삭제 권한 구현
- [ ] 개인정보 처리 동의 절차
