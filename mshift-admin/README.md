# Archived Root Files (Remaining)

git-filter-repo 실행 후에 루트에 남아있던 NextJS 관련 파일들을 정리한 폴더입니다.

## 포함된 파일들

### NextJS 프로젝트 파일들

- `app/` - NextJS App Router 구조
- `components/` - React 컴포넌트들
- `lib/` - 유틸리티 라이브러리
- `public/` - 정적 파일들
- `data/` - 데이터 파일들
- `__tests__/` - Jest 테스트 파일들
- `prisma/` - 데이터베이스 스키마

### 설정 파일들

- `package.json`, `package-lock.json`, `yarn.lock`, `pnpm-lock.yaml` - 의존성
- `next.config.ts` - NextJS 설정
- `tailwind.config.ts` - Tailwind CSS 설정
- `eslint.config.mjs` - ESLint 설정
- `jest.config.js`, `jest.setup.js` - Jest 설정
- `tsconfig.json` - TypeScript 설정
- `components.json` - shadcn/ui 설정
- `middleware.ts` - NextJS 미들웨어
- `postcss.config.mjs` - PostCSS 설정

### 문서 및 데이터 파일들

- `bug.md`, `mvp.md`, `todolist.md`, `룰엔진알고리즘.md` - 문서
- `gyeonggi_data.sql`, `seoul_data_sample.sql`, `supabase_schema_dump.sql` - SQL 파일
- `scripts/` - 유틸리티 스크립트들

## 주의사항

이 파일들은 현재 `mshift-admin/` 폴더에 정리되어 있으므로, 이 아카이브 폴더의 파일들은 백업용입니다.
