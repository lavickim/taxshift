# 소규모 법인 기장 서비스 모바일 앱 PRD

## Executive Summary

본 PRD는 소규모 법인(직원 10명 이하)을 대상으로 하는 AI 기반 자동 기장 서비스 "SmartLedger"의 모바일 앱 개발을 위한 상세 요구사항을 정의합니다. 머니핀의 사용자 친화적 UI와 자비스의 법인 특화 기능을 벤치마킹하여, 하이브리드 룰 엔진(정규식 + LLM)과 Active Learning을 통해 세무 지식이 없는 사용자도 쉽게 사용할 수 있는 혁신적인 기장 솔루션을 제공합니다.

**핵심 차별화 요소:**
- 하이브리드 룰 엔진으로 90% 이상 자동 분류 정확도
- 사용자 피드백 기반 실시간 학습으로 지속적 성능 향상
- 월 59,900원의 경쟁력 있는 가격으로 All-in-One 서비스 제공
- 모바일 중심 설계로 언제 어디서나 5분 내 처리 가능

## 1. 프로덕트 개요 및 목표

### 1.1 비전
"복잡한 법인 회계를 누구나 쉽게, AI가 알아서 처리하는 스마트 기장 서비스"

### 1.2 미션
소규모 법인의 회계 업무 시간을 월 20시간에서 2시간으로 줄이고, 세무사 비용을 90% 절감하여 사업에 집중할 수 있도록 지원

### 3.1 자동 거래 수집 및 분류

#### 3.1.1 하이브리드 룰 엔진
```
기능명: AI 자동 분류 시스템
설명: 정규식 패턴과 LLM을 결합한 2단계 분류 시스템

처리 로직:
1. 1차 분류 (정규식 룰)
   - MCC 코드 기반 카테고리 매칭
   - 거래처명 패턴 인식 (예: *커피 → 복리후생비)
   - 금액 범위 기반 분류
   - 처리 시간: <100ms
   - 정확도: 85%

2. 2차 분류 (LLM)
   - 1차 분류 실패 또는 신뢰도 <80% 케이스
   - GPT-4o 기반 컨텍스트 분석
   - 과거 유사 거래 참조
   - 처리 시간: 1-2초
   - 정확도: 95%

3. 신뢰도 표시
   - 90% 이상: 자동 승인 (녹색)
   - 70-89%: 검토 필요 (노란색)
   - 70% 미만: 수동 분류 (빨간색)
```

### 3.2 사용자 피드백 학습 시스템

#### 3.2.1 Active Learning Pipeline
```python
# 의사 코드
class ActiveLearningSystem:
    def __init__(self):
        self.confidence_threshold = 0.8
        self.feedback_buffer = []
        
    def process_transaction(self, transaction):
        # 1차 룰 엔진 처리
        rule_result = self.rule_engine.classify(transaction)
        
        if rule_result.confidence > 0.9:
            return rule_result
            
        # 2차 LLM 처리
        llm_result = self.llm_engine.classify(transaction, context)
        
        if llm_result.confidence < self.confidence_threshold:
            # 사용자 피드백 요청
            self.request_user_feedback(transaction, llm_result)
            
        return llm_result
        
    def apply_user_feedback(self, feedback):
        # 즉시 학습 반영
        self.update_rule_patterns(feedback)
        self.fine_tune_llm(feedback)
        
        # 유사 거래 재분류
        similar_transactions = self.find_similar(feedback.transaction)
        self.reclassify_batch(similar_transactions)
```

```python
    - 시간: {transaction.timestamp}
    - 설명: {transaction.description}
    
    과거 유사 거래:
    {get_similar_transactions(transaction)}
    
    이 거래를 가장 적절한 계정과목으로 분류하고,
    신뢰도(0-100)와 함께 이유를 설명해주세요.
    """
    
    response = await openai.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "system", "content": ACCOUNTING_EXPERT_PROMPT},
            {"role": "user", "content": prompt}
        ],
        temperature=0.1
    )
    
    return parse_classification(response)
```
