# SmartLedger 모바일앱 개발 계획 및 투두리스트

## 📋 프로젝트 개요

### 목표
- PRD 기반으로 React Native 앱 개발
- 백엔드 API와 연동하여 실제 동작하는 MVP 완성
- 사용자 테스트 가능한 수준의 앱 제작

### 개발 환경
- **플랫폼**: React Native with Expo
- **언어**: TypeScript
- **상태관리**: Redux Toolkit
- **네비게이션**: React Navigation 6
- **UI 라이브러리**: React Native Elements + Custom Components
- **백엔드 연동**: Axios

### 백엔드 API 엔드포인트 요구사항
- `GET /api/dashboard/summary` - 대시보드 요약 데이터
- `GET /api/transactions` - 거래 목록
- `POST /api/transactions` - 거래 추가
- `PUT /api/transactions/{id}` - 거래 수정
- `POST /api/transactions/classify` - AI 분류
- `GET /api/reports/profit-loss` - 손익계산서
- `GET /api/tax/calendar` - 세무 일정
- `POST /api/auth/login` - 로그인
- `GET /api/user/profile` - 사용자 정보
