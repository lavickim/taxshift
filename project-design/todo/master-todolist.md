## 🔧 **TODO 관리 규칙**

### **파일 명명 규칙**
- `{영역명}-todolist.md` 형식 사용
- 영어 소문자와 하이픈(-) 사용
- 예: `mobile-bookkeeping-todolist.md`

### **상태 표시 규칙**
- `[ ]` - 미완료 작업
- `[x]` - 완료된 작업  
- `✅` - 완전히 검증된 작업
- `🔄` - 진행 중인 작업
- `⏸️` - 일시 중단된 작업
- `❌` - 취소된 작업

### **우선순위 표시**
- `🔴` - High Priority (긴급 & 중요)
- `🟡` - Medium Priority (중요)
- `🟢` - Low Priority (일반)
- `⚪` - Info (정보성)

### **업데이트 주기**
- **일일**: 개별 TODO 파일 업데이트
